Trabalho feito pela equipe VatapaueR, neste site montamos uma plataforma de visualização e analise de dados de forma dinâmica.

Com a grande quantidade de dados do mundo moderno, se faz necessário plataformas que facilitem a análise desses para a tomada de decisões de forma correta por parte de empresas e governos.

Como esses dados estão constatemente sendo atualizados, uma plataforma dinâmica é imprescindível.
